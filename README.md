# 🌈 Caterpillars Playschool & Daycare

A colorful, adorable and fully responsive preschool website built for **Caterpillars Playschool & Daycare** 🧸✨

Designed with a magical garden-inspired theme to create a warm, joyful and welcoming experience for parents and children.

---

## 🌟 Features

- 🌈 Beautiful preschool-inspired UI
- 📱 Fully responsive (mobile + desktop)
- 🦋 Animated floating clouds & magical decorations
- 📸 Interactive activity gallery with image lightbox
- 💛 Emotional and parent-friendly design
- 📍 Google Maps location section
- 💬 WhatsApp quick contact button
- 🎨 Modern colorful branding
- ⚡ Fast static website

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Responsive Design
